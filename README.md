# Premium Portfolio Website Template

A modern, high-end portfolio website boilerplate built with Vanilla HTML, CSS, and JavaScript.

## Features
- **Premium Aesthetics**: Dark mode theme with electric indigo accents.
- **Glassmorphism**: Modern UI elements with backdrop filters.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Smooth Animations**: Intersection Observer-driven fade-in effects.
- **SEO Optimized**: Semantic HTML and meta tags ready for search engines.

## Setup
1. Open `index.html` in any modern web browser.
2. Customize the content in `index.html`.
3. Update your styles in `style.css`.
4. Add logic or extra interactions in `main.js`.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Inter & Outfit)
